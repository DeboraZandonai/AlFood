
# Alfood

O Alfood é um site que lista restaurantes e pratos do menu. 

## ✔️ Técnicas e tecnologias utilizadas

Se liga nessa lista de tudo que usaremos nesse treinamento:

- `React`
- `React Hooks`
- `TypeScript`
- `axios`
